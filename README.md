## 🎨 CSS ART GALLERY

An interactive CSS Art Showcase Web App built using HTML, CSS, and JavaScript, featuring handcrafted CSS artwork, category filtering, favorites system, live source viewing, interactive tweaks, and immersive gallery exploration.

## 🚀 Features
* **16 Handcrafted CSS Artworks** – Explore unique artworks built entirely using pure CSS shapes and animations.
* **Category Filtering System** – Filter artworks by Nature, Characters, Abstract, Pixel Art, and Favorites.
* **Real-Time Search** – Instantly search artworks by title.
* **Favorites System** – Save favorite artworks for quick access later.
* **Dark/Light Theme Toggle** – Switch between editorial dark and light themes.
* **Dynamic Stats Bar** – Displays total visible artworks and favorite counts.
* **Interactive Artwork Cards** – Hover animations and dynamic previews for each artwork.
* **Detailed Modal View** – Open each artwork in an immersive full-screen modal.
* **CSS Technique Tags** – View techniques used in each artwork.
* **Source Code Viewer** – View HTML and CSS source code for every artwork.
* **CSS Challenges Section** – Learn by attempting challenge prompts for each artwork.

## 🎮 How to Use
**Browse Gallery**

**Search Artwork**

* Clear search instantly using reset button

**Filter Categories**
* Filter by Nature
* Filter by Characters

**Save Favorites**
* Click heart icon on artwork cards
* Access favorite artworks later

**Open Artwork Details**
* Click any artwork card
* View full artwork preview
* Read artwork description
